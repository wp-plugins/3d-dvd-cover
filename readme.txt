3D DVD 
It is a plugin for modeling dvd covers threejs WebGL and it can generate outstanding images to a post.

== Description ==

Have you ever tried to add a 3D model web without the need of looking for a software to generate such a model?, Well, 3d dvd is part of a set of plugins for adding 3D models and generate images from that model png format, this is part of an online project developed by <a href="http://all.develoteca.com/builder/ ">http://all.develoteca.com/builder/</a>  and <a href="http://develoteca.com/ ">Develoteca.com</a> team.
